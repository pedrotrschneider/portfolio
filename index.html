O que eu faço

Design e desenvolvimento de jogos

Desde 2018 desenvolvo jogos por diversão. Comecei aprendendo sozinho durante o ensino médio, mas, depios de me apaixoar pela área, decidi buscar esino superior na USP em Ciência da Computação para aprofundar meus conhecimentos, especialmente na área de computação gráfica.

Design de narrativa para jogos

Dentro de desenvolvimento de jogos, uma área que sempre me encantou foi o design de narrativas. A grande expressividade e interatividade presente em jogos tem um excelente potencial para contar ótimas histórias

Desenvolvimento de software

Além de jogos, também sou desenvolvedor de software no geral, englobando tanto ferramentas de linha de comando para sistemas Linux, com Bash e Rust, quanto desenvolvimento de aplicações para Desktop e Web, como este site.

Quem eu sou

Designer e desenvolvedor de jogos desde 2018

Comecei a desenvolver jogos em 2018, focando na parte técnica do desenvolvimento. Meu primeiro jogo foi feito em Java, com uma engine feita por mim mesmo, utilizando OpenGL. Rapidamente percebi que meu aspecto preferido do desenvolvimento de jogos era a parte criativa e visual dos jogos, e, com isso em mente, fui buscar formas de acelerar meu desenvolvimento.

Para isso, comecei a utilizar a engine Godot, e permaneço com ela até hoje, por sua simplicidade, praticidade, e, claro, por ser sfotware livre. Quando não estou utilizando a Godot, Rust é minha linguagem de escolha para programação no geral.

Também desenvolvo outros tipos de software, com programas de linha de comando para Linux e aplicações de desktop, além de desenvolvimento web, como este site.

Atualmente, sou estudante do Bacharelado em Ciência da Computação na Universidade de São Paulo, cursando o quinto período.
